# SwitchballHD-Level-Editor
This is a Blender addon, which helps with custom level creation for Switchball, it uses blender to transform object data into .vnl data which is essentially a file structured for ingame and main menu levels.


For proper usage you can refer to a [video here](https://youtu.be/9-HiA-3076o):

and here is the publication on [Nexus Mods](https://www.nexusmods.com//mods/1?game_id=7227), where I will keep it up to date over there, while the files in the latest release are just used to download and get you started, actual updates and changes to the mod as a whole will be easily accessible on Nexus Mods compared to over here.
